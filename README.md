# MRjvm

My own implementation of jvm.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mrjvm'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mrjvm 

## Usage

Just build and install mrjvm. Then type mrjvm [file]
